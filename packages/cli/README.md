# robrain CLI

Shared memory across your team and AI agents, with structured rejected alternatives and corpus-wide judgment.

- Main docs: https://github.com/adelinamart/robrain#readme
- CLI reference: https://github.com/adelinamart/robrain/blob/main/docs/cli.md
- Concepts: https://github.com/adelinamart/robrain/blob/main/docs/concepts.md
- Troubleshooting: https://github.com/adelinamart/robrain/blob/main/docs/troubleshooting.md

## Quick start (self-hosted)

```bash
npx robrain@latest up
npx robrain install --self-hosted
cd /path/to/your/project
npx robrain init-project
```

Fully-local LLM (Ollama / LM Studio / vLLM, no cloud keys):
https://github.com/adelinamart/robrain/blob/main/docs/cli.md#fully-local-llm-ollama--lm-studio--vllm
